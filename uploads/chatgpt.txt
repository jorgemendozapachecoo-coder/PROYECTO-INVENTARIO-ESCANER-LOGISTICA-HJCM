### Paso a Paso del Proyecto de Gestión de Inventario con CRUD y Exportación

Voy a describir paso a paso el flujo completo de tu aplicación de gestión de inventario, asegurando que se entienda cómo funcionan los últimos códigos funcionales de CRUD (Agregar, Editar, Eliminar) y la exportación de datos. Esto te permitirá comprender mejor la integración de las diferentes partes del sistema.

#### 1. Inicio de Sesión y Registro
- **Inicio de sesión (`login.php`)**: Los usuarios deben iniciar sesión con sus credenciales para acceder a la aplicación. Si el usuario no está autenticado, es redirigido a esta página.
- **Verificación de sesión**: Cada archivo PHP, como `inventory.php`, `add_product.php`, `edit_product.php`, y `delete_product.php`, verifica si el usuario ha iniciado sesión. Si no, lo redirige a `login.php`.

#### 2. Visualización del Inventario (`inventory.php`)
- **Carga de Productos**: Al ingresar a `inventory.php`, se carga la lista completa de productos desde la base de datos. Se muestra una tabla con todos los productos, incluyendo columnas como `ITEM`, `NOMBRE_AREA`, `CODIGO_FAMILIA_ACTUAL`, `DENOMINACION_BIEN`, `MARCA`, `MODELO`, `SERIE`, `FECHA_ADQUISICION`, `CODIGO_PATRIMONIAL`, etc.
- **Búsqueda de Productos**: Se proporciona un campo de búsqueda para buscar productos mediante el `CODIGO_PATRIMONIAL`. Si el producto es encontrado, se resalta en la tabla.
- **Botones de Acciones**: Cada producto tiene opciones para **Editar** y **Eliminar**:
  - **Editar**: El botón de editar redirige al usuario a `edit_product.php`, enviando el ID del producto mediante `GET`.
  - **Eliminar**: El botón de eliminar envía el ID del producto mediante `POST` a `delete_product.php`, para eliminarlo de la base de datos.

#### 3. CRUD Completo
- **Agregar Producto (`add_product.php`)**:
  - El formulario permite agregar un nuevo producto al inventario.
  - Antes de agregar el producto, se verifica que el `CODIGO_PATRIMONIAL` no exista ya en la base de datos.
  - Si el producto es agregado correctamente, el usuario es redirigido a `inventory.php` con un mensaje de éxito.

- **Editar Producto (`edit_product.php`)**:
  - Cuando el usuario hace clic en el botón de "Editar" en `inventory.php`, se redirige al archivo `edit_product.php` con el ID del producto.
  - Los datos actuales del producto se cargan en el formulario de edición para que el usuario pueda modificarlos.
  - Al actualizar, se verifica si el `CODIGO_PATRIMONIAL` ha cambiado. Si se ha cambiado, se verifica que no sea duplicado.
  - Si la edición es exitosa, el usuario es redirigido a `inventory.php` con un mensaje de éxito.

- **Eliminar Producto (`delete_product.php`)**:
  - Cuando se hace clic en el botón de eliminar, se envía una solicitud `POST` a `delete_product.php` con el ID del producto.
  - Se verifica si el producto existe antes de eliminarlo.
  - Si se elimina correctamente, el usuario es redirigido a `inventory.php` con un mensaje de éxito.

#### 4. Importación y Exportación de Inventario
- **Importar Inventario (`import_inventory.php`)**:
  - Los usuarios pueden importar un archivo Excel con los productos del inventario. Los encabezados deben coincidir con los campos del inventario para que los datos se carguen correctamente.
  - Si el producto ya existe (basado en el `CODIGO_PATRIMONIAL`), se actualiza con la nueva información.

- **Exportar Inventario (`export_inventory.php`)**:
  - Los usuarios pueden exportar el inventario completo a un archivo Excel.
  - Los productos con `ANTIGÜEDAD` mayor a 10 años se resaltan en rojo para indicar que necesitan revisión o reemplazo.

#### 5. Flujo General del Proyecto
1. **Inicio de Sesión**: El usuario ingresa con sus credenciales.
2. **Visualización del Inventario**: Se muestran todos los productos con las opciones para buscar, editar, eliminar o agregar productos.
3. **Agregar Producto**: Se puede agregar un nuevo producto al inventario.
4. **Editar Producto**: Se selecciona un producto existente para modificar su información.
5. **Eliminar Producto**: Se elimina un producto del inventario.
6. **Importar/Exportar Inventario**: Se pueden importar nuevos productos desde un archivo Excel o exportar el inventario actual.

Este flujo garantiza que los usuarios puedan gestionar su inventario de manera eficiente, con la capacidad de agregar, editar, eliminar, buscar, importar y exportar productos. La aplicación está diseñada para ser intuitiva y fácil de usar, proporcionando mensajes claros al usuario para cada acción realizada.
